# Android Project by Can Luo, Yao Tong

This repository contains simple COVID19 data monitoring with android stack MVVM, Live Data, Koin, RxJava, RxBinding, Offline first with simple caching, etc

2. Koin
3. Retrofit
4. RxBinding

9. Spek2 Framework
